# 📘 Matemáticas para Computadores

## 📌 Descripción
Este repositorio contiene materiales y ejercicios relacionados con la asignatura de **Matemáticas para Computadores**.  

El objetivo principal es aplicar conceptos matemáticos fundamentales en el ámbito de la informática, como los sistemas numéricos y algoritmos básicos utilizados en programación.

---

## 🗂️ Estructura del proyecto
El repositorio incluye distintos archivos y prácticas relacionadas con la materia:

- 📄 Archivos de código
- 📄 Ejercicio a resolver
- 📄 Explicacion de como se resuelve

---

## ⚙️ Contenido
Entre los contenidos que se pueden encontrar en este repositorio destacan:

- ➕ Algoritmo de para encontrar maximos o minimos Simplex y Simplex dual 
- 🧠 Uso de estrategias de Hiper plano y branch and bound
- 📊 Resolución de ejercicios prácticos

---

## 🚀 Uso

1. Clonar el repositorio:
```bash
git clone https://github.com/RicardoSotoGallo/MatematicasComputadores.git
```bash
2. Inicie el proyecto desde la carpeta MateMaticas
3. Inserte el ejercicio en el fichero : MateMaticas/Ficheros/ecuacion.txt
4. Ejuecute el fichero main en: MateMaticas/out/production/MateMaticas/org/example/Main.class
5. Bebe agua
